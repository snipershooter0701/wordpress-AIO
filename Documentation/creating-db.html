    Create Database

    1. Open MySQL® Database Wizard
    2. Enter a database name and click to Next Step
    3. Set username and password then click to Create User You can use Password Generator for creating strong passwords. But be sure the password does not contain any special characters, choose only numbers. Copy the password to clipboard.
    4. Add user to the database. Check All Privileges option then click to Make Changes
    5. You will see a confirmation alert after database and user created successfully.
